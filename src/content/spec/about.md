# AI부업노트

AI부업노트는 집에서 해볼 만한 AI 부업을 직접 화면으로 확인하고, 해볼 사람 입장에서 정리하는 블로그입니다.

데이터라벨링처럼 단순 반복 작업에 가까운 일은 일부러 덜 다룹니다. 대신 애드센스 블로그, 전자책 판매, 숏폼 대본, 블로그 원고 대행, 바이브 코딩처럼 실제 산출물이 남는 일을 봅니다. 화면을 열어보고, 어떤 순서로 시작하면 좋을지 남깁니다.

## 운영 기준

- 한국인이 접근하기 쉬운 플랫폼과 도구를 우선합니다.
- 과장된 수익 문구보다 준비물, 작업 시간, 실패 가능성을 같이 적습니다.
- 정보만 나열하지 않고, 직접 확인한 화면과 내 판단을 같이 남깁니다.

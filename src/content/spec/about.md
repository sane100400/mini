# AI부업노트

AI부업노트는 집에서 시도할 수 있는 AI 기반 부업을 기록하는 블로그입니다.

데이터라벨링처럼 단순 반복 작업에 가까운 일은 제외하고, 애드센스 블로그, 전자책 판매, 숏폼 제작, 블로그 원고 대행처럼 실제 산출물이 남는 일을 중심으로 정리합니다.

## 운영 기준

- 한국인이 접근하기 쉬운 플랫폼과 도구를 우선합니다.
- 과장된 수익 문구보다 준비물, 작업 시간, 실패 가능성을 같이 적습니다.
- 직접 실행 가능한 체크리스트와 작업 흐름을 남깁니다.

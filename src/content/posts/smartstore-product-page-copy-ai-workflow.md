스마트스토어 판매자 중에는 사진은 준비했는데 문구에서 막히는 사람이 많다. 제품 설명을 길게 써도 구매자가 궁금한 내용이 빠져 있으면 상세페이지는 힘을 쓰지 못한다. 이 지점은 AI를 잘 쓰면 재택 부업으로 만들기 좋다.

다만 "상세페이지를 전부 만들어드립니다"로 시작하면 일이 커진다. 초보는 문구 개선부터 잡는 편이 현실적이다. 사진 배치와 디자인까지 맡기면 수정이 끝없이 늘어난다.

![상품 촬영을 준비하는 책상](./images/product-photo.jpg)

## 먼저 고객 질문을 뽑는다

상품 문구는 장점 나열이 아니다. 구매자가 결제 전에 묻는 질문에 답하는 일이다. 예를 들어 텀블러라면 보온 시간, 세척 방법, 무게, 뚜껑 밀폐, 배송 중 파손 여부가 중요하다.

AI에게 바로 "상세페이지 문구 써줘"라고 시키면 흔한 문장이 나온다. 먼저 질문부터 뽑게 한다.

| 상품 | 먼저 확인할 질문 |
|---|---|
| 의류 | 소재, 핏, 세탁, 사이즈 |
| 식품 | 원재료, 보관, 배송, 유통기한 |
| 생활용품 | 사용법, 내구성, 교환 기준 |
| 전자기기 | 호환성, 구성품, 보증 기간 |

질문이 정리되면 문구는 훨씬 빨리 나온다. 판매자가 이미 가진 사진과 상세 설명에서 답을 찾고, 부족한 부분은 다시 물어보면 된다.

![온라인 판매 자료를 노트북으로 정리하는 모습](./images/ecommerce-desk.jpg)

## 팔기 좋은 납품물

초반 상품은 짧게 쪼개는 것이 좋다. 상세페이지 전체를 맡는 대신 아래처럼 납품물을 나눈다.

- 첫 화면 후킹 문구 5개
- 제품 장점 문장 10개
- FAQ 문구 8개
- 상세페이지 섹션 제목 6개
- 구매 전 확인 문구 5개

이 정도면 디자인을 몰라도 팔 수 있다. 의뢰인은 기존 상세페이지에 문구를 붙여 넣을 수 있고, 작업자는 사진 편집에 시간을 빼앗기지 않는다.

## 사람 손이 들어가야 하는 부분

AI는 장점을 잘 부풀린다. 그래서 최종 문구에서는 과장 표현을 걷어내야 한다. "최고의", "완벽한", "압도적인" 같은 말은 실제 근거가 없으면 빼는 편이 낫다. 판매 페이지에서는 멋진 말보다 구체적인 말이 더 오래 간다.

![배송과 상품 판매를 상징하는 장바구니 이미지](./images/shopping-cart.jpg)

작업 전에 판매자에게 받아야 할 자료도 정해둔다. 상품 사진, 원재료나 소재, 배송 정책, 교환 기준, 자주 받은 문의가 필요하다. 이 다섯 가지가 없으면 문구가 예뻐도 빈말이 된다.

스마트스토어 문구 개선은 화려한 부업은 아니다. 대신 고객이 명확하다. 사진을 올렸는데 판매가 안 되는 사람, 상세페이지를 만들었지만 말이 약한 사람에게 제안하기 좋다. AI는 초안을 빠르게 만들고, 사람은 과장과 빈칸을 잡는다. 그 역할을 분명히 하면 집에서도 시작해볼 만하다.

## 스마트스토어 글에도 실제 수요 화면을 넣는다

스마트스토어 판매자에게 제안할 부업이라면 실제 마켓 수요 화면을 봐야 한다. 스마트스토어 판매자센터는 자동화 캡처가 정상적으로 잡히지 않아 이번 글에는 넣지 않았다. 대신 크몽 공개 화면에서 쇼핑몰 운영자가 찾는 서비스 영역을 확인한다. 아래 화면에는 "쇼핑몰 사장님이 많이 찾아요" 영역과 검색 최적화, 상세페이지 같은 키워드가 보인다.

![크몽 공개 메인 화면. 쇼핑몰 운영자가 찾는 검색 최적화와 상세페이지 서비스가 보인다.](./images/kmong-marketplace-screen.jpg)

이 화면이 중요한 이유는 단순하다. 상세페이지 문구 개선은 실제로 시장에서 검색되는 작업이다. 다만 "스마트스토어 상세페이지 전부 제작"으로 들어가면 디자인, 이미지, 촬영, 기획까지 일이 커진다. 초보는 문구 개선, FAQ, 첫 화면 후킹 문구처럼 작게 잘라야 한다.

## 판매자에게 제안할 작은 상품

처음 팔기 좋은 상품은 아래 정도다.

| 상품 | 납품물 | 필요한 자료 |
|---|---|---|
| 첫 화면 문구 | 후킹 문구 5개, 서브 문구 5개 | 상품명, 대표 장점, 타깃 |
| FAQ 정리 | 구매 전 질문 10개와 답변 | 배송, 교환, 소재, 사용법 |
| 상세페이지 제목 | 섹션 제목 8개 | 기존 상세페이지 캡처 |
| 장점 문장 | 제품 장점 15문장 | 상세 설명, 사진 |
| 전후 비교 | 기존 문구 5개 수정 | 판매자가 쓰던 문구 |

이 상품들은 디자인을 몰라도 가능하다. 판매자는 기존 상세페이지에 문구를 붙여 넣으면 된다. 작업자는 AI로 초안을 만들고, 사람이 과장과 빈칸을 잡는다.

## 상세페이지 문구는 질문에서 시작한다

상품 문구를 장점으로 시작하면 흔해진다. 먼저 고객 질문을 뽑는다.

| 상품군 | 구매 전 질문 |
|---|---|
| 의류 | 사이즈가 작게 나오는지, 세탁하면 줄어드는지 |
| 식품 | 유통기한, 보관 방법, 배송 중 파손 여부 |
| 생활용품 | 세척 방법, 내구성, 구성품 |
| 전자기기 | 호환 기기, 보증 기간, 구성품 |
| 디지털 상품 | 파일 형식, 사용 방법, 환불 기준 |

이 질문에 답하면 상세페이지가 구체적이 된다. "고급스러운 디자인"보다 "찬물 세탁 가능", "오후 2시 전 주문 당일 출고", "PDF와 Notion 링크 제공" 같은 문장이 더 강하다.

## AI 초안에서 지워야 할 말

상세페이지 문구에서 AI가 자주 쓰는 말은 아래다.

| AI식 표현 | 바꾸는 방향 |
|---|---|
| 최고의 품질 | 어떤 소재와 공정을 썼는지 |
| 누구나 만족 | 어떤 사람에게 맞는지 |
| 완벽한 선택 | 어떤 상황에서 쓰는지 |
| 특별한 경험 | 실제 사용 장면 |
| 합리적인 가격 | 구성품과 비교 기준 |

예를 들어 "누구나 만족하는 텀블러"보다 "출근길 500ml 커피를 담기 좋고, 차량 컵홀더에 들어가는 지름입니다"가 낫다. 상세페이지는 멋진 문장보다 구매 전 불안을 줄이는 문장이 필요하다.

## 문구 개선 전후 예시

포트폴리오에는 아래처럼 전후 비교를 넣는다.

| 기존 문구 | 수정 문구 | 이유 |
|---|---|---|
| 고급 소재로 제작했습니다 | 겉감은 생활 방수가 되는 폴리에스터이며, 안감은 물티슈로 닦을 수 있습니다 | 소재와 관리 방법을 구체화 |
| 선물용으로 좋습니다 | 기본 종이 포장에 리본 색상 2가지를 선택할 수 있습니다 | 선물 상황을 실제 옵션으로 설명 |
| 편리하게 사용할 수 있습니다 | 손잡이 폭이 넓어 장갑을 낀 상태에서도 들기 쉽습니다 | 편리함을 행동으로 바꿈 |

이 정도 샘플 5개면 포트폴리오가 된다. 판매자는 작업자의 문장 감각보다 "이 사람이 내 상품의 빈칸을 찾는구나"를 본다.

## 작업 전에 받을 자료

상세페이지 문구 개선은 자료가 없으면 빈말이 된다. 시작 전에 아래 자료를 받아야 한다.

| 자료 | 확인할 것 |
|---|---|
| 기존 상세페이지 | 어떤 정보가 이미 있는지 |
| 상품 사진 | 문구로 설명할 디테일 |
| 배송 정책 | 출고 시간, 택배사, 제주·도서산간 |
| 교환·환불 기준 | 구매 전 불안 해소 |
| 자주 받은 문의 | FAQ 문구의 재료 |

자료를 받지 못하면 "임시 문구"로 납품한다고 적는다. 그래야 나중에 사실 확인 책임이 작업자에게 몰리지 않는다.

## 블로그 글로 확장하기

스마트스토어 문구 개선은 블로그 시리즈로 만들기 좋다.

- 상세페이지 첫 화면 문구 5개 고치는 법
- 배송 문구를 구매 전 불안 해소 문장으로 바꾸는 법
- AI 상세페이지 문구에서 과장 표현 걷어내기
- FAQ 문구 10개를 먼저 만들면 상세페이지가 쉬워지는 이유
- 크몽에 상세페이지 문구 개선 상품을 올릴 때 주의할 점

이런 글은 애드센스에도 맞고, 실제 의뢰 포트폴리오에도 맞다. 글을 읽은 판매자가 바로 자기 상세페이지를 떠올리게 만드는 것이 목표다.

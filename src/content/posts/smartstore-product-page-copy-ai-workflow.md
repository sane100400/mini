스마트스토어 상세페이지 문구 개선은 AI 부업으로 시작하기 좋은 편이다. 상품 사진, 가격, 배송 정보, 옵션은 이미 있는데 설명만 약한 판매자가 많기 때문이다. 이때 필요한 것은 예쁜 카피보다 고객이 실제로 묻는 질문이다.

다만 "AI로 예쁘게 써드립니다"보다 "후기와 문의를 보고 문구표로 정리합니다"가 더 설명하기 쉽다. 판매 문구는 사실과 연결되어야 한다. 처음 상품은 후기와 문의에서 반복되는 질문을 상품명, 핵심 문구, FAQ로 바꾸는 정도가 적당하다.

| 작업 범위 | 시작하기 좋은가 | 이유 |
| --- | --- | --- |
| 상품명 후보 5개 | 추천 | 범위가 작고 비교가 쉬움 |
| 핵심 문구 10개 | 추천 | 상세페이지 첫 화면에 쓰기 좋음 |
| 고객 질문 기반 문장 5개 | 추천 | 구매 전 궁금증을 줄임 |
| FAQ 5개 | 추천 | 후기·문의에서 바로 뽑을 수 있음 |
| 배너 디자인 전체 | 나중에 | 디자인 수정 범위가 커짐 |
| 상세페이지 전체 리뉴얼 | 나중에 | 사진, 배치, 디자인, 검수까지 커짐 |

## 수요 화면에서 작업 위치를 본다

상세페이지 문구 개선을 부업으로 보려면 실제 판매자가 쓰는 화면을 먼저 확인해야 한다. 상단 대표 이미지는 네이버 스마트스토어센터 화면을 직접 캡처했다. 이 화면을 보면 상품 등록과 운영 흐름 안에서 문구 개선이 어디에 들어가는지 생각하게 된다.

판매자는 보통 "문구만 필요하다"고 말하지 않는다. 상품이 잘 안 팔린다, 상세페이지가 허전하다, 후기에서 같은 질문이 반복된다, 광고 클릭은 있는데 구매가 안 된다는 식으로 말한다. 이 말을 문구 작업으로 바꾸려면 고객 질문을 찾아야 한다.

| 상품 | 고객 질문 |
| --- | --- |
| 텀블러 | 보온 시간, 세척 방법, 무게, 밀폐, 컵홀더 호환 |
| 반려동물 샴푸 | 성분, 향, 피부 자극, 사용 주기, 용량 |
| 운동복 | 사이즈, 착용감, 세탁 후 변형, 비침 |
| 전자책 | 대상 독자, 분량, 업데이트, 예시 파일 |

## 처음에는 문구표로 납품한다

처음에는 상세페이지 전체 리뉴얼보다 문구 개선만 작은 상품으로 제안하는 편이 좋다. 디자인, 사진, 배너, 옵션표, 리뷰 구성까지 모두 다루지 않아도 고객 질문을 정리한 문구표만으로 충분히 도움이 된다.

| 납품물 | 형식 |
| --- | --- |
| 상품명 후보 | 표 |
| 핵심 문구 | 문서 또는 표 |
| 고객 질문 기반 문장 | FAQ 형태 |
| 수정 전후 비교 | 2열 표 |
| 확인 필요 표시 | 메모 칸 |

처음에는 디자인 파일을 납품하지 않는 쪽이 낫다. 텍스트 문서나 표로 납품하고, 판매자가 직접 스마트스토어에 반영하거나 디자이너에게 넘기도록 정리한다. 작업 범위가 작으면 낮은 가격으로 시작하고, 수정 범위도 관리하기 쉽다.

## 문구는 고객 질문에서 시작한다

상세페이지 문구를 고칠 때 처음 보는 것은 문장의 예쁨이 아니다. 고객이 어떤 순서로 궁금해하는지다. 상품을 처음 본 사람은 이게 나에게 맞는지, 가격이 납득되는지, 불편한 점은 없는지, 배송과 교환은 괜찮은지 확인한다.

| 기존 문구 | 문제 | 고친 문구 |
| --- | --- | --- |
| 일상에서 편리하게 사용할 수 있는 텀블러입니다 | 너무 넓음 | 350ml 용량이라 출근 가방에 넣기 부담이 적습니다 |
| 부드러운 착용감입니다 | 근거가 약함 | 목 뒤 봉제선을 안쪽으로 접어 오래 입어도 덜 거슬립니다 |
| 실용적인 제품입니다 | 장면이 없음 | 차량 컵홀더에 들어가는 바닥 지름으로 만들었습니다 |

단, 실제 정보가 있을 때만 쓴다. 수치와 기능은 판매자가 확인한 자료 안에서만 쓰면 된다. 상세페이지 문구는 예쁜 말보다 확인 가능한 정보가 먼저다.

## AI 초안에서 지울 말

AI 초안에는 자주 나오는 말이 있다. "최고의", "프리미엄", "혁신적인", "효율적인", "편리한", "만족도를 높입니다" 같은 표현이다. 실제 근거가 있으면 써도 되지만, 대부분은 넓다.

| 지울 가능성이 큰 말 | 바꿀 방향 |
| --- | --- |
| 최고의 품질 | 어떤 소재, 어떤 구조인지 |
| 프리미엄 제품 | 가격이 납득되는 이유 |
| 편리한 사용감 | 어떤 상황에서 덜 불편한지 |
| 만족도를 높입니다 | 어떤 불만을 줄이는지 |

AI가 만든 문장을 고칠 때는 세 단계만 거쳐도 좋아진다. 먼저 사실과 맞지 않는 말을 지운다. 다음으로 고객 질문에 답하지 않는 문장을 지운다. 마지막으로 너무 긴 문장을 둘로 나눈다.

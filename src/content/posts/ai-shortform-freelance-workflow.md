AI 숏폼 제작은 재택 부업으로 시작하기 쉬워 보인다. 결과물이 눈에 보이고, 대본·자막·썸네일·편집 파일처럼 일을 나누기도 쉽기 때문이다. 외주 상품으로 만들 때는 이 단위를 잘게 나눠두면 설명이 편해진다.

처음에는 20초 대본부터 시작하는 편이 좋다. 자막 문장과 썸네일 문구까지는 초보도 어디까지 해줄지 설명하기 쉽고, 의뢰인도 결과물을 검수하기 쉽다. 이후 편집안, 완성 영상, 업로드 보조로 넓히면 된다.

| 가능한 일 | 시작하기 좋은가 | 이유 |
| --- | --- | --- |
| 20초 대본 | 좋음 | 범위가 작고 수정 기준이 분명함 |
| 자막 문장 | 좋음 | 화면용 문장을 짧게 다듬는 작업 |
| 썸네일 문구 | 좋음 | 여러 후보를 빠르게 만들 수 있음 |
| 편집안 | 보통 | 컷 순서와 자막 위치까지만 제안 |
| 완성 영상 | 나중에 | 수정, 음원, 저작권, 파일 관리가 커짐 |

## 처음에는 대본부터 판다

완성 영상은 할 일이 많다. 소재 조사, 대본, 컷 편집, 자막, 템플릿, 음원, 썸네일, 내보내기, 업로드 비율 확인까지 들어간다. 한 단계만 어긋나도 수정 시간이 늘어난다. 반면 대본은 범위가 작다. 첫 3초 후킹, 본문 전개, 마지막 행동 유도만 정하면 된다.

| 대본 상품명 | 전달할 내용 |
| --- | --- |
| 제품 소개용 20초 숏폼 대본 5개 | 후킹, 본문, CTA |
| 전자책 홍보용 릴스 대본 3개 | 문제 제기, 해결 기준, 구매 연결 |
| 동네 매장 이벤트 쇼츠 문구 10개 | 첫 화면 문구, 짧은 내레이션 |
| 블로그 유입용 쇼츠 대본 5개 | 글 주제 요약, 저장 유도 |

AI는 소재를 여러 각도로 뽑고, 첫 문장을 여러 버전으로 만들고, 자막 문장을 짧게 줄이는 데 쓴다. 초안이 나오면 화면에 올라갈 문장과 말로 읽을 문장을 나눠서 한 번 더 다듬는다.

## CapCut 화면을 보면 작업이 나뉜다

상단 대표 이미지는 CapCut 공식 홈 화면을 직접 캡처했다. AI 기반 사진·영상 편집기라는 메시지와 온라인 사용 버튼이 보인다. 이 화면만 봐도 대본 뒤에 편집, 자막, 템플릿 작업이 이어진다는 걸 알 수 있다.

| 결과물 | 설명 | 초보 추천 |
| --- | --- | --- |
| 대본 | 말로 읽을 문장 | 추천 |
| 자막 문장 | 화면에 올라갈 짧은 문장 | 추천 |
| 썸네일 문구 | 클릭 전 보이는 제목 후보 | 추천 |
| 편집안 | 장면 순서와 자막 위치 | 상황에 따라 |
| 완성 영상 | 실제 편집 파일 | 나중에 |

CapCut은 편집까지 가능하게 해준다. 그래서 대본, 자막 문장, 썸네일 문구를 먼저 정리한 뒤 편집으로 넓히면 일이 덜 흔들린다.

## Shorts는 수익보다 형식부터 본다

YouTube Creators의 Shorts 안내 화면도 같이 봤다. 화면은 짧은 영상 제작과 수익화를 강조한다. 프리랜서 관점에서는 수익보다 영상 형식이 먼저다.

20초 대본은 아래처럼 쪼개면 다루기 쉽다.

| 시간 | 쓰임 | 예시 |
| --- | --- | --- |
| 0-3초 | 멈추게 하는 문장 | AI 부업 글감이 매번 막힌다면 |
| 3-10초 | 상황 설명 | 키워드만 보면 글이 다 비슷해집니다 |
| 10-17초 | 해결 기준 | 먼저 검색자가 막힌 지점을 표로 적습니다 |
| 17-20초 | 행동 유도 | 체크리스트는 블로그에 정리해뒀습니다 |

## 업종별 샘플부터 만든다

포트폴리오가 없을 때는 업종별 샘플을 만든다. 실제 고객을 기다릴 필요는 없다. 전자책, 온라인 클래스, 스마트스토어, 동네 매장, 개인 브랜딩처럼 있을 법한 상황을 정하고 대본을 만든다.

| 업종 | 좋은 시작 문장 | 덜 맞는 문장 |
| --- | --- | --- |
| 전자책 | AI 부업 글을 쓰려는데 제목에서 막힌다면 | 쉽고 빠르게 돈 버는 법 |
| 스마트스토어 | 비 오는 날 운동화가 젖는 게 싫다면 | 최고의 품질을 자랑합니다 |
| 동네 매장 | 퇴근 후 30분만 비는 사람에게 맞춘 클래스 | 누구나 만족하는 서비스 |
| 개인 브랜딩 | 첫 의뢰를 준비하는 포트폴리오가 필요하다면 | 전문가처럼 보이게 해드립니다 |

숏폼 문장은 짧아야 한다. "본 제품은 일상생활에서 편리하게 활용할 수 있습니다"보다 "가방 안에서 바로 꺼내 쓰기 쉽습니다"가 낫다. 한 화면에는 한 메시지만 남긴다.

## 수정 범위를 적어두면 작업이 편해진다

숏폼 작업에서는 수정 범위를 미리 적어두는 편이 좋다. 의뢰인이 대본만 고치려는 것인지, 영상 편집까지 바꾸려는 것인지 구분하기 쉬워진다. 그래서 상품 설명에 포함과 제외를 계약서처럼 적어둔다.

| 상품 | 포함 | 제외 |
| --- | --- | --- |
| 대본 상품 | 제목, 후킹, 본문 대사, CTA | 영상 편집, 업로드 |
| 자막 문장 상품 | 화면용 짧은 문장 | 컷 편집, 음원 |
| 편집안 상품 | 컷 순서, 자막 위치 제안 | 실제 편집 파일 |
| 완성 영상 상품 | 편집 파일, 자막, 내보내기 | 원본 촬영, 광고 집행 |

완성 영상까지 맡는다면 원본 영상 제공 방식, 자막 스타일, 음원 사용, 썸네일, 수정 횟수, 전달 파일 형식, 세로 비율, 저작권 책임을 적어야 한다. 이 내용이 있으면 작은 금액의 작업도 깔끔하게 끝난다.

## 블로그 글로 확장한다

숏폼 부업은 애드센스 블로그 글감으로도 이어진다. 작업하면서 나오는 질문이 많기 때문이다.

| 글감 | 넣을 근거 |
| --- | --- |
| CapCut으로 편집할 때 자막을 줄이는 법 | CapCut 화면, 자막 전후 비교 |
| YouTube Shorts와 Reels 대본 차이 | 포맷 표, 예시 대본 |
| 제품 소개 숏폼 대본 5개 만드는 순서 | 업종별 샘플 |
| 숏폼 대본 외주에서 수정 범위를 적는 법 | 포함·제외 표 |

AI 숏폼 제작은 집에서 시작해도 된다. 다만 처음부터 완성 영상을 맡기보다 작게 나눈 결과물부터 다루는 편이 안전하다. 대본을 먼저 팔고, 자막 문장과 썸네일 문구를 더하고, 편집안으로 넓힌 뒤 완성 영상으로 가면 된다.
